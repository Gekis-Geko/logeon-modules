# Logeon Combat Admin Tools

Estensione opzionale Tier 3 che dipende da `logeon.narrative-combat`.

## Scopo

Il modulo introduce:

- dashboard staff sui contesti combattimento
- diagnostica di escalation, code pending e partecipanti a rischio
- pannello staff nel conflict hub con hotspot contestuali

## Dipendenze

- `logeon.narrative-combat` obbligatorio
- `Tier 2` richiesto come profondita minima effettiva del modulo base
