# Logeon Combat AI

Estensione opzionale Tier 3 che dipende da `logeon.narrative-combat`.

## Scopo

Il modulo introduce un primo layer di:

- profili comportamentali per partecipante
- suggerimenti tattici contestuali
- dichiarazione staff-guidata delle azioni suggerite

## Dipendenze

- `logeon.narrative-combat` obbligatorio
- `Tier 2` richiesto come profondita minima effettiva del modulo base

## Stato attuale

Questa prima release copre:

- authoring admin dei profili AI per conflitto/personaggio
- suggerimenti visibili nel pane Tier 3 del combat hub
- dichiarazione rapida dell'azione suggerita lato staff

Non copre ancora:

- esecuzione autonoma in background
- reasoning multi-turn complesso
- apprendimento o adattamento storico
