# Logeon Combat Coordination

Estensione opzionale Tier 3 che dipende da `logeon.narrative-combat`.

## Scopo

Il modulo introduce:

- piani coordinati per leader e supporter
- bonus reali al resolver quando la manovra e coerente
- consumazione automatica del piano su risoluzione

## Dipendenze

- `logeon.narrative-combat` obbligatorio
- `Tier 2` richiesto come profondita minima effettiva del modulo base

## Stato attuale

Questa prima release copre:

- authoring admin dei piani
- creazione e annullamento dal pane game
- bonus al punteggio di risoluzione per leader e supporter
- consumo automatico del piano quando il leader completa la manovra prevista
